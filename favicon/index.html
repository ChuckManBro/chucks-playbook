favicon demo

You should be seeing an American Flag 'favorite icon' on the browser tab.

Udemy tutorial shows only the following link in the HTML head:
link rel="icon" href="favicon.ico"

VSCode snippet produces all this:
link rel="shortcut icon" href="favicon.ico" type="image/x-icon"

If favicon is not updating, put this into href:
"favicon.ico?v=2"
